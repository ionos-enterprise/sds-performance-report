IBNBD Performance Evaluation

IBNBD (InfiniBand Network Block Device) is a linux kernel driver enabling for accessing a remote block device over RDMA network. We developed IBNBD and IBTRS (InfiniBand transport library) as an open source project with low IO latency as the main design criteria. While other open source solutions, such as SRP or NVMEoF do provide similar functionality, IBNBD is showing significantly higher performance in both domains - IO latency and throughput. IBNBD and the underlying general purpose IBTRS library are available for everybody under GPL 2.0 licence.
